In memory database and API

This is a placeholder page. To run queries go to the /query endpoint

You can override this placeholder page by adding this to your SimplyStore server code:

 
	import SimplyStore from '@muze-nl/simplystore'

	SimplyStore.get('/', function(req, res) {
	        res.setHeader('Content-Type', 'text/html')
	        res.send('Your homepage content')
	})

	SimplyStore.run()
